# Vitor Refrigeracao Website

CodeIgniter 4 project with Docker-based local development (Apache + PHP-FPM + MySQL).

## Quick Start

```bash
yarn install
docker compose up -d --build
```

App URLs:

- `http://localhost:8080/`
- `http://vitorrefrigeracao.local:8080/` (if mapped in `/etc/hosts`)
